# Proyecto: CreditSmart - Interfaz de Usuario

Nombre del estudiante: Emanuel Gonzalez Grisales 

Descripción breve
------------------
Pequeña aplicación estática para mostrar un catálogo de productos crediticios, un simulador y un formulario de solicitud. El proyecto es una maqueta (frontend estático) pensada para practicar maquetación, estilos CSS y navegación entre páginas.

Estructura de archivos
----------------------
```
Credit-Smart_IU/
├─ index.html             # Página principal (Catálogo)
├─ simulador.html         # Página del simulador
├─ solicitar.html         # Página para solicitar crédito
├─ css/
│  ├─ common.css         # Estilos comunes (variables,header, footer, botones)
│  ├─ index.css          # Estilos específicos de la página index (catálogo)
│  ├─ simulador.css      # Estilos específicos del simulador
│  └─ solicitar.css      # Estilos específicos del formulario de solicitud
├─ images/                # Imágenes e íconos usados en las tarjetas
└─ js/                    # (Opcional) scripts JavaScript si se agregan
```

Instrucciones para ejecutar el proyecto
---------------------------------------
1. Abrir el archivo `index.html` en un navegador web (doble clic en el archivo o "Abrir con..." -> navegador).
2. Alternativamente, si usas VS Code puedes instalar la extensión "Live Server" y pulsar "Go Live" para servirlo localmente (recomendado para ver cambios en caliente).
